# 概览仪表盘页面详细设计方案

## 1. 页面概述

### 1.1 页面定位

概览仪表盘是 Salesforce 集成数据中心的首页，用于集中展示系统的核心运行状态、关键指标和实时监控信息。该页面是用户进入数据中心后的第一个页面，需要快速、直观地呈现系统整体健康状况。

### 1.2 设计目标

- **快速洞察**: 用户在 3 秒内了解系统整体状态
- **实时监控**: 关键指标实时更新，及时发现异常
- **直观展示**: 使用可视化元素，降低认知负担
- **可操作性强**: 支持快速操作和跳转

### 1.3 用户场景

- **运维人员**: 每日巡检，监控系统运行状态
- **开发人员**: 查看同步进度和 API 调用情况
- **管理员**: 了解整体使用情况，做出决策

### 10.3 状态管理设计

**使用 Zustand 进行状态管理**:

```typescript
// stores/dashboardStore.ts
import { create } from 'zustand';
import { DashboardData } from '@/pages/Dashboard/types';

interface DashboardState {
  data: DashboardData | null;
  loading: boolean;
  error: string | null;
  lastUpdateTime: Date | null;
  autoRefresh: boolean;
  
  setData: (data: DashboardData) => void;
  setLoading: (loading: boolean) => void;
  setError: (error: string | null) => void;
  setLastUpdateTime: (time: Date) => void;
  setAutoRefresh: (autoRefresh: boolean) => void;
  refreshData: () => Promise<void>;
}

export const useDashboardStore = create<DashboardState>((set, get) => ({
  data: null,
  loading: false,
  error: null,
  lastUpdateTime: null,
  autoRefresh: true,
  
  setData: (data) => set({ data }),
  setLoading: (loading) => set({ loading }),
  setError: (error) => set({ error }),
  setLastUpdateTime: (time) => set({ lastUpdateTime: time }),
  setAutoRefresh: (autoRefresh) => set({ autoRefresh }),
  
  refreshData: async () => {
    const { setData, setLoading, setError, setLastUpdateTime } = get();
    setLoading(true);
    setError(null);
    
    try {
      const response = await fetchDashboardData();
      setData(response);
      setLastUpdateTime(new Date());
    } catch (error) {
      setError(error.message);
    } finally {
      setLoading(false);
    }
  }
}));
```

### 10.4 自动刷新实现

**使用 useEffect 实现自动刷新**:

```typescript
// hooks/useAutoRefresh.ts
import { useEffect, useRef } from 'react';
import { useDashboardStore } from '@/stores/dashboardStore';

export const useAutoRefresh = (interval: number) => {
  const { autoRefresh, refreshData } = useDashboardStore();
  const intervalRef = useRef<NodeJS.Timeout | null>(null);
  
  useEffect(() => {
    if (autoRefresh) {
      intervalRef.current = setInterval(() => {
        refreshData();
      }, interval);
    } else {
      if (intervalRef.current) {
        clearInterval(intervalRef.current);
      }
    }
    
    return () => {
      if (intervalRef.current) {
        clearInterval(intervalRef.current);
      }
    };
  }, [autoRefresh, interval, refreshData]);
};
```
